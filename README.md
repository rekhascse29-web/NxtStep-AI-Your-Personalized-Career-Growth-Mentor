# NxtStep AI

## Problem
Students struggle to identify their skill gaps, choose the right career path, and know what to learn next.

## Solution
NxtStep AI analyzes student information using AI and generates:
- Skill Gap Analysis
- Career Recommendation
- Personalized Learning Roadmap
- Resume Tips
- Interview Preparation

## Features
- AI Career Analysis
- Skill Gap Detection
- Career Prediction
- Personalized Learning Roadmap
- Resume Suggestions
- Interview Preparation

## Tech Stack

### Frontend

- CSS
- JavaScript

### Backend
- Python
- Flask

- Google Gemini API
